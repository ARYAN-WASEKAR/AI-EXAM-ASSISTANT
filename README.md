# 🤖 AI Exam Assistant

## 📌 Overview
AI Exam Assistant helps students understand concepts and prepare exam-ready answers. It also evaluates student answers and provides feedback.

---

## 🚀 Features
- 📘 Simple Explanation
- 📝 Bullet Points
- 🎯 2-Mark Answer
- 📖 5-Mark Answer
- 🔑 Keywords
- 🌍 Hindi Translation
- ✅ Answer Feedback System (Score + Suggestions)

---

## 🧠 How It Works
1. User enters question
2. AI generates structured answer
3. User enters answer (optional)
4. AI evaluates and gives feedback

---

## 🛠 Tech Stack
- Google AI Studio (Gemini API)
- Python
- Replit

---

## ▶️ How to Run
1. Add your API key in main.py
2. Run the program
3. Enter question and answer
